# FURNITURE INTAKE OS — PROJECT PROGRESS

**Главная цель:** довести один продаваемый вертикальный продукт для мебельных компаний до работающего пилота.

**Продукт:** сайт + AI-приёмщик + предварительный расчёт + CRM + уведомления + карточка заказа.

**Главный репозиторий продукта:** `Furniture-web-platform-V2`  
**Технологическое ядро:** `furniture-orders-mvp`  
**Первый пилот:** `salamat-mebel.kz`  
**Второй пилот после завершения первого:** `bek-mebel.kz`
